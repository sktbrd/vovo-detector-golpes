### 1. **Verificar Sitemap** (2 min)

No menu lateral:
1. Clique em **"Sitemaps"**
2. Verifique se `sitemap.xml` foi submetido
3. Se NÃO aparecer, adicione:
   - Campo "Add a new sitemap": digite `sitemap.xml`
   - Clique em **"Submit"**
4. Aguarde 1-2 minutos pra processar

**Resultado esperado:** "Success" + começar a ver URLs descobertas

### 2. **Verificar Status de Indexação** (3 min)

No menu lateral:
1. Clique em **"Pages"** (dentro de "Indexing")
2. Veja quantas páginas estão:
   - **Indexed:** Páginas no Google (meta: 22+)
   - **Not indexed:** Páginas descobertas mas não indexadas
   - **Discovered - not indexed:** Google conhece mas não crawlou

**Me manda screenshot ou números!**

Exemplo:
```
Indexed: 0
Not indexed: 22 (Reason: "Discovered - currently not indexed")
```

## 📊 **Verificar Coverage/Performance** (5 min)

### Coverage (Indexação)

No menu lateral:
1. Clique em **"Pages"**
2. Scroll down até "Why pages aren't indexed"
3. Veja quais são os erros (se houver):
   - "Discovered - currently not indexed" → Normal em site novo
   - "Crawled - currently not indexed" → Google crawlou mas não achou relevante (need more content depth)
   - "Redirect error" → Fix necessário
   - "4xx/5xx errors" → Fix crítico

**Me avisa se tiver erros!**

---

### Performance (Tráfego)

No menu lateral:
1. Clique em **"Performance"**
2. Veja se tem algum dado:
   - **Total clicks:** Quantos cliques do Google
   - **Total impressions:** Quantas vezes apareceu na busca
   - **Avg CTR:** % de cliques
   - **Avg position:** Posição média nas buscas

**Resultado esperado:** Provavelmente 0 (site muito novo). Vai aparecer dados em 2-7 dias após indexação.

## 🔧 **Service Account (Opcional - API Access)**

**Se quiser acesso programático via API:**

### Por quê usar Service Account?
- Automatizar checks de indexação
- Pull data pro dashboard
- Monitoramento automático

### Como adicionar:

1. **Criar Service Account** (Google Cloud):
   - Vai em https://console.cloud.google.com
   - Cria projeto (ou usa existente)
   - Ativa "Google Search Console API"
   - Cria Service Account
   - Download JSON key

2. **Adicionar no Search Console:**
   - Clique em **"ADD USER"** (botão azul que aparece na screenshot)
   - Email: `service-account-email@project-id.iam.gserviceaccount.com`
   - Permission: **Full** (ou **Owner** se quiser controle total)
   - Clique em **"Add"**

**Nota:** Isso NÃO é necessário agora. Só se você quiser automatizar depois.
