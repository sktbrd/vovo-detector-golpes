# Como Enviar Mensagem Criptografada Segura e Autodestrutível 🔐

Precisa compartilhar uma senha, número de cartão ou informação confidencial pela internet? Enviar dados sensíveis por WhatsApp, Telegram ou e-mail é **extremamente arriscado** — essas mensagens ficam gravadas nos servidores e podem ser acessadas por terceiros. A solução? **Mensagens criptografadas que se autodestroem** após a leitura.

## O que é uma Mensagem Criptografada Autodestrutível? 🤔

Uma mensagem criptografada autodestrutível é uma forma de comunicação digital onde o conteúdo é:
- **Criptografado** com algoritmos militares (AES-GCM 256-bit)
- **Acessível apenas uma vez** — após ser lida, é destruída permanentemente
- **Protegido por chave única** — só quem tem o link pode acessar
- **Sem rastreamento** — a chave de descriptografia fica no link, nunca é enviada ao servidor

### Por Que Usar Mensagens Autodestrutíveis? 📊

- **Compartilhar senhas temporárias**: Envie credenciais de Wi-Fi, acessos a sistemas, senhas de uma vez
- **Dados financeiros sensíveis**: Números de cartão, códigos de segurança, chaves PIX
- **Informações confidenciais**: CPF, RG, contratos, documentos sigilosos
- **Comunicação segura**: Conversas que não devem deixar rastros permanentes

## Como Funciona a Criptografia AES-GCM 256-bit? 🔒

O padrão AES-GCM 256-bit é o mesmo usado por governos e militares. Para ter uma ideia da segurança:
- **Número de combinações possíveis**: 2^256 (mais de 10^77)
- **Tempo para quebrar**: Mais de 1 trilhão de anos usando os computadores mais potentes do mundo
- **Certificações**: NSA, NIST, ISO 27001

Quando você envia uma mensagem criptografada:
1. Sua mensagem é convertida em código indecifrável
2. Uma chave única é gerada e fica no link (após o #, que o servidor nunca vê)
3. Quando alguém acessa o link, a mensagem é descriptografada **no navegador**
4. Após a leitura, o servidor deleta permanentemente a mensagem

## Perguntas Frequentes (FAQ) ❓

### A mensagem fica salva em algum lugar?
Não. Após ser lida, a mensagem é permanentemente deletada do servidor. Não há backups ou recuperação possível.

### E se eu precisar enviar a mesma mensagem para várias pessoas?
Você precisa criar um link para cada pessoa. Cada mensagem é única e se autodestrói na primeira leitura.

### Posso ver se alguém já leu a mensagem?
Não. O sistema é zero-knowledge — nem mesmo o servidor sabe se alguém leu ou não.

### É realmente seguro enviar senha de cartão por aí?
Com mensagens criptografadas autodestrutíveis, sim. A chave nunca é enviada ao servidor (fica no link após o #), e a mensagem é destruída após 1 leitura. É **infinitamente mais seguro** que WhatsApp, e-mail ou SMS.
